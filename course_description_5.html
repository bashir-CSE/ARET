Course Description

"বর্তমান সময়ে উদ্ভাবনী গবেষণায় দক্ষ হওয়া খুবই জরুরি। তাই বর্তমান প্রজন্মকে উদ্ভাবনী গবেষণায় দক্ষ করে তুলতে এবং আপনার মাইক্রোকন্ট্রোলার শেখার যাত্রা আরো সহজ করতে আমার নিতে এসেছি ARDUINO TRAINER KIT।
এখন পর্যন্ত সারা বিশ্বে মাইক্রোকন্ট্রোলার হিসাবে আরডুইনো ব্যবহারকারীর সংখ্যা সবচেয়ে বেশি। ফলে আরডুইনো-কোম্পাটিবল সেন্সর এবং হার্ডওয়্যার গুলো তুলনামূলক সহজলভ্য এবং সাশ্রয়ী।"

What You'll Learn

  • Arduino programming fundamentals

Trainer Kit Components

Hardware Components
  • Arduino Neno board
  • Eight LEDs Interfacing
  • Seven Segment Display Interfacing
  • Liquid Crystal Display (LCD) Interfacing
  • Ultrasonic Sensor Interface
  • LDR Interface
  • Relay Interface
  • ADC Interfacing
Additional Components
  • Push Button Interfacing
  • Infrared Interfacing
  • Bluetooth Interface
  • Reed switch interface
  • Temperature sensor LM35 interfacing
  • Digital temperature and humidity sensor DHT 22
  • External Supply Socket (5V)
  • External Power rail +5Vdc and GND
