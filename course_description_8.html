NI LabVIEW Programming

Course Overview

Master LabVIEW programming for test automation and measurement systems

15 Days 10 Seats BDT 7000.00

Course Description

"LabVIEW is a graphical programming environment that provides unique productivity accelerators for test system development, such as an intuitive approach to programming, connectivity to any instrument, and fully integrated user interfaces."

Prerequisites:

Basic understanding of electrical systems, automation, and control systems.

Course Syllabus

Course Information

  • Course Title NI LabVIEW Programming
  • Duration 15 Days
  • Total Seats 10 Seats
  • Course Fee BDT 7000.00

What You'll Learn

  • LabVIEW programming fundamentals
  • Arduino integration with LabVIEW
  • Digital and analog I/O control
  • Sensor data acquisition and visualization
  • PWM control for motors and LEDs
  • Test automation and measurement systems

Course Syllabus

  • Overview of LabVIEW and Arduino integration
  • Installing the LINX Toolkit for LabVIEW
  • Setting up communication between LabVIEW and Arduino
  • Creating a simple program to blink an LED using LabVIEW

Objective: Set up the environment and establish basic communication between LabVIEW and Arduino.

  • Controlling Arduino's digital pins via LabVIEW
  • Turning LEDs on/off from LabVIEW using digital output
  • Reading digital inputs (e.g., push buttons) in LabVIEW
  • Creating an interactive interface to control multiple LEDs and read button states

Objective: Control digital I/O pins and interact with LabVIEW's graphical interface.

  • Reading analog inputs from sensors (e.g., potentiometer, temperature sensor) via Arduino
  • Displaying real-time sensor data in LabVIEW
  • Visualizing data using LabVIEW charts and indicators
  • Calibrating sensor values and applying simple data processing in LabVIEW

Objective: Interface analog sensors with Arduino and display sensor data in real-time in LabVIEW.

  • Using PWM to control the brightness of LEDs via LabVIEW
  • Controlling the speed of DC motors with PWM signals from Arduino
  • Implementing sliders in LabVIEW to adjust PWM duty cycle in real time
  • Experimenting with different PWM frequencies for smooth control

Objective: Learn to control PWM-enabled devices like LEDs and motors using LabVIEW.
